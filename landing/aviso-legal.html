2

Objeto y naturaleza del proyecto

TUSSAM es una aplicación móvil independiente creada con el objetivo de ofrecer una experiencia de usuario mejorada para la consulta de los autobuses urbanos de Sevilla. Nace como un proyecto para mejorar la usabilidad respecto a la aplicación oficial de la ciudad.

La aplicación permite consultar en tiempo real los tiempos de llegada de los autobuses, localizar paradas cercanas, buscar líneas y rutas, y guardar paradas favoritas. Está disponible para iPhone y Apple Watch.

Este proyecto no tiene ánimo de competir con el servicio oficial, sino de complementarlo ofreciendo una interfaz más accesible y cómoda para los usuarios del transporte público de Sevilla.

3

Fuentes de datos y servicios GIS

La aplicación obtiene la información sobre paradas, líneas y tiempos de llegada a partir de fuentes de datos públicas proporcionadas por el servicio de transporte urbano de Sevilla.

Para la localización de paradas y la representación cartográfica, la aplicación utiliza servicios de Sistemas de Información Geográfica (GIS) y los servicios de mapas nativos de Apple (MapKit). Los datos geográficos de las paradas provienen de fuentes públicas abiertas.

  • Los datos de tiempos de llegada se consultan en tiempo real desde fuentes públicas del servicio de transporte.
  • La información geográfica de paradas (coordenadas, nombres, direcciones) se obtiene de datos abiertos.
  • La representación en mapa utiliza Apple MapKit con datos cartográficos de Apple.
  • No se accede a ninguna API privada ni se vulnera ningún sistema de acceso restringido.

Todos los datos utilizados son de acceso público. Si algún titular de datos considera que su uso es indebido, rogamos contacte con legal@sparring.dev para resolverlo a la mayor brevedad.
